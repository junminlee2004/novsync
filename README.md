# novsync
A TAIHEN plugin for automatically disabling vsync on game startup on a per game basis

Based on Electry's VGi functionality for disabling vsync

This plugin functions like any TAIHEN plugin in terms of installation

To install this plugin, copy the novsync.suprx file to ur0:tai/ folder on your modded vita, 

then enter your config file and add the following lines corresponding to your game title so that the plugin will launch with the game

e.g. for Uncharted:

*PCSA00029

ur0:tai/novsync.suprx

RESTART YOUR VITA
